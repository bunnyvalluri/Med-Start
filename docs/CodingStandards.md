# Coding Standards & Guidelines — MedStart

## 1. Principles
- **Strict TypeScript**: `noImplicitAny: true`, `strictNullChecks: true`.
- **Component Design**: Pure functional React components with explicit prop types.
- **Error Handling**: Graceful fallback states for every asynchronous operation.
- **Naming Conventions**: PascalCase for React components (`HospitalCard.tsx`), camelCase for utility functions (`calculateDistance.ts`), kebab-case for filenames.
