# Product Roadmap — MedStart

## Phase 1: MVP Core Platform & Emergency Discovery (Completed)
- Hospital search by city, name, specialty, and emergency status.
- Interactive Leaflet map with GPS geolocation and turn-by-turn routing.
- RBAC Admin Dashboard for hospital & doctor CRUD management.
- Firebase Auth & Cloud Firestore data layer setup.

## Phase 2: AI-Powered Triage & Real-Time Ambulance Tracking (Upcoming)
- Integration of Tele-health appointment booking.
- Real-time IoT bed sensor sync for emergency rooms.
- Live ambulance GPS dispatch tracking.
